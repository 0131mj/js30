# 11 Custom Video Player



### Video tag

```html
<video src="url"></video>
```

- 영상을 재생시켜 주는 태그 
- 자바스크립트를 사용해 메서드를 실행하거나, 프로퍼티를 변경하여 관련 기능을 제어할 수 있다.  



### Video Method

```javascript
video.play();  // 재생
video.pause(); //일시정지
```

- 콘솔창에서 메서드를 실행해보자. 



### Video Properties

```javascript
video.src 	= "./video";// 소스주소
video.muted = true; 	// 무음여부 설정(true,false)
video.volume = 1; 		// 볼륨비율 (0~1)
video.currentTime = 10; // 재생할 시간지점
```

- 콘솔창에서 프로퍼티를 수정해보자.
